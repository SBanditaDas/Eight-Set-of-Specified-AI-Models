## 🚀 Overview

**AI Model Explorer** is a cutting-edge, interactive dashboard designed to showcase the power of modern AI architectures. From Large Language Models to specialized Vision and Action agents, this platform provides a unified interface to experiment with the latest advancements in Artificial Intelligence.

Whether you're exploring generative art, complex reasoning, or real-time object segmentation, AI Model Explorer offers a premium, high-performance experience powered by industry-leading engines.

---

## 🧠 Supported Architectures

Explore 8+ distinct AI model families, each optimized for specific tasks:

| Type | Name | Description |
| :--- | :--- | :--- |
| **LLM** | Large Language Model | Advanced text generation, coding, and complex reasoning. |
| **VLM** | Vision Language Model | Multimodal understanding—ask questions about any image. |
| **LCM** | Latent Consistency Model | Ultra-fast, consistent image generation in milliseconds. |
| **LAM** | Large Action Model | Conversational agents that plan and execute tool-based tasks. |
| **SLM** | Small Language Model | Efficient, low-latency models optimized for edge performance. |
| **MoE** | Mixture of Experts | Sparse architecture activating specialized "expert" sub-networks. |
| **MLM** | Masked Language Model | Bidirectional context understanding by predicting hidden tokens. |
| **SAM** | Segment Anything Model | Zero-shot object segmentation and spatial localization. |

## 🔌 Dual-Provider Engine

The explorer features a seamless switching mechanism between two powerful AI backends:

1.  **Google Gemini**: Industry-leading multimodal performance, native function calling, and high-fidelity image analysis.
2.  **Open Source (Hugging Face)**: Access to the world's most popular open-weight models including Llama 3, Phi-3, BLIP, and DETR.

## 💻 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- API Keys for Gemini and/or Hugging Face

### Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/ai-model-explorer.git
    cd ai-model-explorer
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Environment Setup**:
    Create a `.env` file in the root directory and add your credentials:
    ```env
    VITE_GEMINI_API_KEY=your_gemini_api_key_here
    VITE_HF_TOKEN=your_huggingface_token_here
    ```

4.  **Launch the Dashboard**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser.
